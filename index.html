Quick-Start Guide
Get Windows games running in 5 steps
1
Check your Snapdragon GPU
Go to Settings → About phone and note your SoC model (e.g. Snapdragon 8 Elite). Match it to the compatibility table above — A8xx users get the latest Turnip ZIP automatically.
2
Download Winlator + latest Turnip driver
Grab the Winlator APK of your choice below, and for A8xx devices use the featured driver button that directly fetches the newest .zip from GitHub.
3
Install Winlator and import driver
Install the APK, open Winlator, go to Settings → GPU Driver, tap Import, and select the downloaded Turnip .zip. Then set it as the active driver.
4
Configure DXVK and container
Create a new container. Under Graphics, select Turnip as renderer and enable DXVK. For DX12 games, also enable VKD3D-Proton in the container settings.
5
Add your game and launch
Tap Add Shortcut, browse to your game's .exe, assign it to the container, and hit Start. Use GameHub for a clean front-end experience with cover art.
